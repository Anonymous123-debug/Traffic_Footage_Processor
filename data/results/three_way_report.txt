================================================================================
KEY FINDINGS
================================================================================

1. EDGE PARALLEL (5 workers):
   - Best throughput for multi-camera deployments
   - Total CPU: 7.0% (combined from 5 workers)
   - Requires multiple edge devices

2. HYBRID ADAPTIVE (5 edge + 1 cloud):
   - Best balance of performance and accuracy
   - 55.3% faster than Cloud
   - 40.0% less CPU than Cloud
   - 43.5% less RAM than Cloud
   - Intelligent video-based routing (70% edge, 30% cloud)
   - Both edge and cloud resources available

3. CLOUD ONLY (1 worker):
   - Baseline performance
   - Highest resource usage
   - Network dependent

================================================================================
RECOMMENDATIONS
================================================================================

USE HYBRID MODE WHEN:
  • Have multiple edge devices + cloud available
  • Need balance between performance and accuracy
  • Want intelligent video-based load distribution
  • Can accept 27% cloud processing for complex videos
  • Want fault tolerance (cloud backup for edge)

USE EDGE PARALLEL WHEN:
  • Multiple edge devices available
  • Maximum throughput needed
  • Minimal cloud dependency desired
  • Network bandwidth is limited

USE CLOUD WHEN:
  • High accuracy is critical
  • Unlimited network bandwidth
  • Edge devices not available
  • Need centralized processing
